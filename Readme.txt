Click ที่ index.thml เพื่อเริ่มเล่น
ในเกมจะมีปุ่ม Start new game เพื่อเริ่มเล่น

เมื่อกดเล่นเกมแล้วจะมีมาเพิ่มอีก 4 ปุ่ม
คือ 
1.Attack เพื่อโจมตี
2.Special Attack โจมตีแรงกว่าปกติ
3.Heal เพิ่มเลือดให้กับตัวเรา
4.Give Up ยอมแพ้เพื่อเริ่มใหม่

เมื่อเล่นจนจบเกม(ไม่ว่าจะชนะหรือแพ้)จะมี Alert ถามถึงการเล่นต่อ
